Optional Server-side Mods
=========================

These mods are optional mods for minecraft servers, or for playing single player.
The mods do not need to be installed on clients as well.
Drop any wanted mods into your `.minecraft\mods` folder.

* AdvancedMotd - Provides additional information in server listing. By default, the time of day.
* DiscordIntegration - Very briefly, connects minecraft chat with Discord chat.
* LagGoggles - Allows players to profile tick lag and fps lag, the visualize it in the world.
* worldedit-forge - WorldEdit allows various powerfull editing features.
